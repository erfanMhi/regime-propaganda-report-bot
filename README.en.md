# Regime Propaganda Report Bot

Report accounts connected to the Islamic regime's propaganda machine on Instagram and X.

Many anti-regime protesters have been wrongfully killed. Propaganda networks label them "terrorists" instead of the heroes they were. This tool helps report those accounts.

---

## Download

**[Download ZIP](https://github.com/erfanMhi/regime-propaganda-report-bot/archive/refs/heads/main.zip)**

Works on Chrome, Edge, Brave (any Chromium browser) on Windows, Mac, Linux.

---

## Installation

1. Download and extract the ZIP file
2. Open Chrome ‚Üí `chrome://extensions`
3. Enable **Developer mode** (top-right toggle)
4. Click **Load unpacked** ‚Üí select the `extension` folder

Done. If you don't see the icon, click the puzzle icon in Chrome's toolbar and pin the extension.

---

## Usage

### Instagram

1. Open **instagram.com** and log in
2. Click the extension icon ‚Üí select **Instagram** tab
3. Click **Start**

The bot reports each account for "False information" with 15-25 second intervals.

### X (Twitter)

1. Open **x.com** and log in
2. Click the extension icon ‚Üí select **X** tab
3. Click **Start**

The bot reports each account for "Violent & hateful entities" with 20-30 second intervals.

---

## Adding Targets

Edit the text box in the popup. One username per line, without @:

```
username1
username2
username3
```

The pre-filled list is a starting point. Add or remove accounts as you see fit. If you notice someone incorrectly included, please let us know.

---

## Daily Limit

The bot pauses after 200 reports per day to help prevent account restrictions. This is conservative: I've done 500+ in a day without issues. Click "Continue Anyway" to override.

# 🚨 ربات گزارش پروپاگاندا رژیم

**گزارش حساب‌های مرتبط با ماشین تبلیغاتی رژیم اسلامی در اینستاگرام و X (توییتر).**

---

## ⬇️ دانلود

**[📦 دانلود ZIP](https://github.com/erfanMhi/regime-propaganda-report-bot/archive/refs/heads/main.zip)**

**کار می‌کند روی:** Chrome، Edge، Brave (هر مرورگر Chromium) — ویندوز، مک، لینوکس

---

## ⚡ نصب (۲ دقیقه)

### مرحله ۱: دانلود و استخراج

1. روی **دانلود ZIP** بالا کلیک کنید
2. فایل دانلود شده را از حالت فشرده خارج کنید (Extract)
3. داخل آن یک پوشه به نام `extension` می‌بینید

### مرحله ۲: بارگذاری افزونه

1. کروم را باز کنید و در نوار آدرس تایپ کنید: `chrome://extensions`
2. دکمه **Developer mode** را روشن کنید (گوشه بالا سمت راست)
3. روی دکمه **Load unpacked** کلیک کنید (بالا سمت چپ)
4. پوشه `extension` را پیدا کنید و انتخاب کنید
5. روی **Select Folder** کلیک کنید

✅ تمام! افزونه نصب شد.

> **آیکون را نمی‌بینید؟** روی آیکون پازل 🧩 در نوار ابزار کروم کلیک کنید، سپس روی پین 📌 کنار "Instagram Report Bot" کلیک کنید

---

## 🚀 نحوه استفاده

### برای اینستاگرام

1. یک تب جدید باز کنید و به **instagram.com** بروید
2. با نام کاربری و رمز عبور وارد شوید
3. روی **آیکون افزونه** در نوار ابزار کلیک کنید
4. تب **Instagram** را انتخاب کنید
5. لیست حساب‌های پروپاگاندا را می‌بینید (از قبل پر شده)
6. روی دکمه سبز **Start** کلیک کنید
7. ربات:
   - به هر پروفایل می‌رود
   - آن‌ها را برای «اطلاعات نادرست» گزارش می‌کند
   - ۱۵ تا ۲۵ ثانیه بین گزارش‌ها صبر می‌کند

### برای X (توییتر)

1. یک تب جدید باز کنید و به **x.com** (یا twitter.com) بروید
2. با حساب خود وارد شوید
3. روی **آیکون افزونه** در نوار ابزار کلیک کنید
4. تب **X** را انتخاب کنید
5. یک لیست اولیه از حساب‌های شناسایی شده را می‌بینید (می‌توانید آن را تغییر دهید یا حساب‌های جدید اضافه کنید)
6. روی دکمه **Start** کلیک کنید
7. ربات:
   - به هر پروفایل می‌رود
   - آن‌ها را برای «موجودیت‌های خشونت‌آمیز و نفرت‌پراکن» گزارش می‌کند
   - ۲۰ تا ۳۰ ثانیه بین گزارش‌ها صبر می‌کند

### تمام!

کروم را باز نگه دارید. هر وقت خواستید روی **Stop** کلیک کنید.

## ❓ رفع مشکلات

| مشکل | راه حل |
|---------|----------|
| "Instagram Tab: Not Found" | مطمئن شوید instagram.com در یک تب باز است و لاگین هستید |
| "X Tab: Not Found" | مطمئن شوید x.com یا twitter.com در یک تب باز است و لاگین هستید |
| آیکون افزونه دیده نمی‌شود | روی آیکون پازل 🧩 کلیک کنید → افزونه را پین کنید |
| گزارش‌دهی وسط کار متوقف شد | ممکن است UI پلتفرم تغییر کرده — آپدیت‌ها را چک کنید |
| با کلیک Start هیچ اتفاقی نمی‌افتد | تب پلتفرم را رفرش کنید و دوباره امتحان کنید |

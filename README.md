## ⚡ شروع سریع

### مرحله ۱: دانلود و اجرا

فایل ZIP پروژه را دانلود کنید، **از حالت فشرده خارج کنید**، سپس فایل اجرا را باز کنید — اگر پایتون یا کروم نصب نباشد، **خودکار** صفحه دانلود باز می‌شود.

| مک | ویندوز |
|-----|---------|
| دابل کلیک روی `Start Bot.command` | دابل کلیک روی `Start Bot.bat` |

یک پنجره مرورگر با رابط ربات باز می‌شود.

### مرحله ۲: تنظیم کروم

1. **اول کروم را کاملاً ببندید** (Cmd+Q در مک / بستن همه پنجره‌ها در ویندوز)
2. در رابط ربات روی **«اجرای کروم»** کلیک کنید
3. **وارد اینستاگرام شوید** در پنجره کروم *(فقط یک‌بار — جلسه ذخیره می‌شود)*

### مرحله ۳: شروع گزارش‌دهی

در رابط ربات روی **«شروع»** کلیک کنید — تمام!

## 📝 اضافه کردن اهداف

فایل `data/targets.txt` را ویرایش کنید — هر نام کاربری در یک خط (بدون @):

```
username1
username2
username3
```

## ❓ رفع مشکلات

| مشکل | راه حل |
|---------|----------|
| "Python not found" | پایتون را دوباره نصب کنید، گزینه "Add to PATH" را تیک بزنید |
| ربات اجرا نمی‌شود | مطمئن شوید کروم نصب است |
| Permission denied (مک) | در ترمینال اجرا کنید: `chmod +x "Start Bot.command"` |

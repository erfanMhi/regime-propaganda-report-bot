# 🚨 ربات گزارش پروپاگاندا رژیم

**گزارش حساب‌های مرتبط با ماشین پروپاگاندا جمهوری اسلامی.**

**چرا؟** بسیاری از معترضان ضد رژیم به ناحق کشته شده‌اند. شبکه‌های پروپاگاندا آن‌ها را «تروریست» می‌نامند، نه قهرمانانی که بودند. این ابزار کمک می‌کند راحت‌تر این حساب‌ها را گزارش کنید.

---

## ⬇️ دانلود

**[📦 دانلود ZIP](https://github.com/erfanMhi/regime-propaganda-report-bot/archive/refs/heads/main.zip)**

**کار می‌کند روی:** Chrome، Edge، Brave (هر مرورگر Chromium) — ویندوز، مک، لینوکس

---

## ⚡ نصب (۲ دقیقه)

### مرحله ۱: دانلود و استخراج

1. روی **دانلود ZIP** بالا کلیک کنید
2. فایل دانلود شده را از حالت فشرده خارج کنید (Extract)
3. داخل آن یک پوشه به نام `extension` می‌بینید

### مرحله ۲: بارگذاری افزونه

1. کروم را باز کنید و در نوار آدرس تایپ کنید: `chrome://extensions`
2. دکمه **Developer mode** را روشن کنید (گوشه بالا سمت راست)
3. روی دکمه **Load unpacked** کلیک کنید (بالا سمت چپ)
4. پوشه `extension` را پیدا کنید و انتخاب کنید
5. روی **Select Folder** کلیک کنید

✅ تمام! افزونه نصب شد.

> **آیکون را نمی‌بینید؟** روی آیکون پازل 🧩 در نوار ابزار کروم کلیک کنید، سپس روی پین 📌 کنار "Instagram Report Bot" کلیک کنید

---

## 🚀 نحوه استفاده

### مرحله ۱: ورود به اینستاگرام

1. یک تب جدید باز کنید
2. به **instagram.com** بروید
3. با نام کاربری و رمز عبور وارد شوید

### مرحله ۲: شروع گزارش‌دهی

1. روی **آیکون افزونه** در نوار ابزار کلیک کنید
2. لیست حساب‌های پروپاگاندا را می‌بینید (از قبل پر شده)
3. روی دکمه سبز **Start** کلیک کنید
4. ببینید که ربات:
   - به هر پروفایل می‌رود
   - آن‌ها را برای "Violence / Calling for violence" گزارش می‌کند
   - ۱۵ تا ۲۵ ثانیه بین گزارش‌ها صبر می‌کند

### مرحله ۳: تمام!

کروم را باز نگه دارید. هر وقت خواستید روی **Stop** کلیک کنید.
